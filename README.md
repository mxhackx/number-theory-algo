# 📘 Number Theory Algorithms

A small collection of fundamental number theory and modular arithmetic algorithms implemented for learning and performance.

## 🚀 Algorithms included

- Fast Exponentiation (Binary Exponentiation)
- Euclidean Algorithm (GCD)
- Extended Euclidean Algorithm
- Sieve of Eratosthenes
- Prime Factorization
- Modular Arithmetic utilities

## 🎯 Purpose

This project is meant to:
- Strengthen algorithmic thinking
- Understand core mathematical building blocks
- Provide efficient implementations for competitive programming and system-level applications

## 🧠 Topics covered

- Number Theory
- Modular Arithmetic
- Prime Numbers
- Optimization techniques

## 💡 Future improvements

- Add segment sieve
- Add modular inverse utilities
- Add more optimized factorization methods
